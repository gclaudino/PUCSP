1 Introdução

Nesta versão, veremos como interromper um fluxo de controle a partir de outro.

Esta versão modifica a maioria das classes desta arquitetura (exceto a Executor). Por meio destas alterações, o usuário poderá enviar eventos do tipo Pausar, cujo tratamento interromperá o fluxo de controle que toca a lista de palavras.

    void definirBotaoTocar(VBox raiz) {
        tocarEvt = new Button("Tocar");
        tocarEvt.setDisable(false);
        tocarEvt.setOnAction(e -> {
            tocarEvt.setDisable(true);
            pausarEvt.setDisable(false);

    }

    // Tocador (v0.6)::
    void tocar(String[] listaPal, long velocidade) {
        int n = listaPal.length;
        for (; p < n; p++) {
            if (parar) {
                // encerra o fluxo que toca palavras
                return;
            }
            mostrar(listaPal[p]);
            suspender(velocidade);
        }
    }
