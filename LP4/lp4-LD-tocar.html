1 Introdução

Nesta versão, faremos uso de um fluxo de controle adicional àquele que controla o tratamento dos eventos do usuário, para executar a lógica de apresentação das palavras. Haverá correção do erro apresentado na versão anterior.

Esta versão adiciona duas novas classes, Adaptador e Executor. Por meio destas novas classes é que será possível separar os dois fluxos de controle da LD.

Listagem 6: Método que define o botão **Tocar**

    // LD (v0.5)::
    void definirBotaoTocar(VBox raiz) {
        Button tocarEvt = new Button("Tocar");
        tocarEvt.setDisable(false);
        tocarEvt.setOnAction(ev -> {
            tocarEvt.setDisable(true);
            // Reação a eventos "Tocar"
            Adaptador a = new Adaptador(t);
            Executor e = new Executor(a);
            e.start();
        });
        raiz.getChildren().add(tocarEvt);
    }
